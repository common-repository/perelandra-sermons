Easily manage your church's sermons with Perelandra Sermons. Upload audio or video files, multiple ways of categorizing, and much more!

== Description ==
Perelandra Sermons is the ultimate sermon manager that allows you to display your sermons, beautifully. Built to integrate seamlessly with WordPress, Perelandra Sermons will help you to broadcast your sermons to an ever widening audience.

= Primary Features =
- Extremely simple settings to get up and running
- Supports both audio and video sermons
- Automatically embed videos from Vimeo, YouTube, and other hosting providers
- Categorize topics by speakers and series
- Automatically display sermons and series on the sermons archive page
- Feature specific sermons and series for the sermons archive
- Add downloadable transcripts for each sermon
- Mobile responsive design for all sermon pages
- Easily customizable templates for your designers & developers



== Installation ==
**Automatic Installation**
Automatic installation is the easiest way to get started with Perelandra Sermons. To automatically install, login to  your WordPress dashboard, navigate to the Plugins menu and click Add New.

Search for "Perelandra Sermons" in the search field and click Install.

That\'s it. You\'re all set. You can start adding sermons to your site by navigating to Sermons in the WordPress admin.

= Manual Installation =
To manually install the plugin, please use the download link on this page to download the zipped plugin files. Using a FTP application, you can follow the instructions on the [WordPress Codex](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation) to upload the files to your server.

= Updating The Plugin =
Updating the plugin should work without any problems. If you do hit a problem, the first thing to do is to flush the permalinks. To do this, login to the WordPress dashboard, navigate to Settings and then Permalinks. Resave the page without making any changes. This should fix any issues that arise when updating.
